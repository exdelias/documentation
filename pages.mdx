Cloudflare Pages is a JAMstack platform for frontend developers to collaborate and deploy websites.

## Key Features

- **Git Integration**: Connect your GitHub or GitLab repository and deploy on every push.
- **Preview Deployments**: Automatically generate preview URLs for every pull request.
- **Fast Performance**: Served from Cloudflare's edge network for lightning-fast loading.
